# Design System

## Theme Strategy

**Drenched (Dark Mode Premium)**
Интерфейс использует глубокий темный тон как основу, создавая кинематографичную и дорогую атмосферу. Основной акцент делается на сами изображения ПК и роскошную типографику, а не на цветные блоки.

## Color Palette (OKLCH)

- **Background (Surface 0):** `oklch(0.12 0 0)` — Истинный глубокий темный (почти черный) для создания безграничного пространства.
- **Surface 1:** `oklch(0.18 0.01 280)` — Слегка приподнятый фон с едва уловимым холодным пурпурным/синим оттенком для карточек и панелей.
- **Ink (Text):** `oklch(0.98 0 0)` — Чистый белый для главных заголовков (высокий контраст).
- **Muted Text:** `oklch(0.70 0.01 280)` — Светло-серый для второстепенного текста, не сливается с фоном (контраст > 4.5:1).
- **Accent:** `oklch(0.70 0.15 280)` — Электрический неоновый пурпурный/синий для кнопок действий и акцентов (High-tech vibe).

## Typography

- **Display & Headings:** `Outfit` или `Inter` (с отрицательным трекингом `-0.02em` для плотности и премиальности).
- **Body:** `Inter` (читаемость).
- **Technical/Mono:** `JetBrains Mono` или `Space Mono` (для инженерных акцентов, цифр "01.", ценников).

## Layout & Components

- **Spacing:** Широкие, дышащие отступы. Контент не должен быть зажат.
- **Cards:** Никаких вложенных карточек! Карточки используются только там, где они действительно нужны, и имеют тончайший бордер (`1px solid oklch(0.25 0 0)`), без заливки, с эффектом `backdrop-filter: blur` (Glassmorphism только для навигации или важных слоев).
- **Motion:** Экспоненциальное затухание (`ease-out-expo`), никаких пружинящих (bounce) анимаций. Переходы должны ощущаться тяжелыми и точными.
