# TOGOSHOL — QA Checklist

## Контент

- [ ] Название бренда: TOGOSHOL.
- [ ] Указан город: Великий Новгород.
- [ ] H1: “Игровые ПК в Великом Новгороде”.
- [ ] Есть акцент на готовые и кастомные ПК.
- [ ] Есть каталог реальных сборок.
- [ ] Есть конфигуратор / подбор.
- [ ] Тексты короткие и продающие.
- [ ] CTA понятные.
- [ ] Нет фейковых отзывов.
- [ ] Нет фейковых контактов.

## Контакты

- [ ] Основной CTA ведёт в рабочий канал связи.
- [ ] VK / Telegram / Max показываются только при наличии реальных ссылок.
- [ ] Телефон показывается только если он реальный.
- [ ] Email показывается только если он реальный.
- [ ] Контакты не дублируются вручную в разных компонентах.
- [ ] VK Market не используется как основной сценарий покупки.

## Структура

- [ ] Header.
- [ ] Hero.
- [ ] Featured Builds.
- [ ] Product Catalog.
- [ ] Product Filters.
- [ ] Product Cards.
- [ ] Configurator.
- [ ] Custom Build.
- [ ] Why TOGOSHOL.
- [ ] Trust Conditions.
- [ ] Order Process.
- [ ] Reviews / Social Proof или секция скрыта.
- [ ] FAQ.
- [ ] Final CTA.
- [ ] Footer.

## Дизайн

- [ ] Фон почти чёрный.
- [ ] Есть cyan / blue / violet glow, но без перегруза.
- [ ] Карточки тёмные, премиальные.
- [ ] Много воздуха.
- [ ] Нет визуальной перегрузки.
- [ ] Типографика крупная и уверенная.
- [ ] Кнопки аккуратные.
- [ ] Badge выглядят чисто.
- [ ] Сайт не выглядит как дешёвый магазин.
- [ ] Сайт не выглядит как пустая декоративная демка.

## Hero

- [ ] Hero выглядит самым сильным экраном.
- [ ] На desktop две колонки или равноценная премиальная композиция.
- [ ] Справа есть визуал ПК или активной сборки.
- [ ] Если нет фото, есть CSS/SVG mockup.
- [ ] Есть 2 CTA.
- [ ] Есть 3 метрики.
- [ ] Текст читается на тёмном фоне.
- [ ] Hero не показывает все товары сразу.

## Каталог

- [ ] Все реальные товары доступны в секции каталога.
- [ ] Есть поиск.
- [ ] Есть фильтр бюджета.
- [ ] Есть фильтр класса сборки.
- [ ] Есть сортировка.
- [ ] Есть кнопка “Показать ещё”.
- [ ] Есть empty state.
- [ ] При смене фильтров каталог не ломается.
- [ ] На mobile карточки идут в одну колонку.

## Карточки Товаров

- [ ] Есть фото или fallback visual.
- [ ] Есть цена.
- [ ] Есть статус.
- [ ] Есть CPU.
- [ ] Есть GPU.
- [ ] Есть RAM.
- [ ] Есть SSD.
- [ ] Есть понятный сценарий / класс сборки.
- [ ] CTA ведёт в контактный канал.
- [ ] Нет грязных строк из импортированного описания.
- [ ] Нет рекламных хвостов из VK.
- [ ] Нет внешней market-ссылки как главного действия.

## Конфигуратор

- [ ] Есть ввод бюджета.
- [ ] Есть выбор игры / задачи.
- [ ] Есть выбор разрешения.
- [ ] Есть выбор RAM.
- [ ] Есть выбор SSD.
- [ ] Есть выбор состояния комплектующих.
- [ ] Есть выбор стиля корпуса.
- [ ] Есть предупреждение для нереалистичного бюджета.
- [ ] Есть 2-3 ближайших товара.
- [ ] Есть кнопка “Скопировать конфигурацию”.
- [ ] Есть CTA “Отправить подбор”.

## Доверие

- [ ] Есть блок “Почему TOGOSHOL”.
- [ ] Есть гарантия.
- [ ] Есть сроки.
- [ ] Есть условия оплаты.
- [ ] Есть выдача / доставка.
- [ ] Есть этапы заказа.
- [ ] Есть отзывы или секция скрыта до появления реальных отзывов.
- [ ] Есть FAQ.

## Анимации

- [ ] Header получает blur при скролле.
- [ ] Hero text появляется через fade-up.
- [ ] Hero visual имеет parallax на desktop.
- [ ] Parallax отключён на mobile.
- [ ] RGB pulse работает плавно и не отвлекает.
- [ ] Cards hover: lift + glow.
- [ ] Buttons hover: scale 1.02.
- [ ] Reveal on scroll работает.
- [ ] Есть prefers-reduced-motion.
- [ ] Анимации не мешают работе с каталогом.

## Адаптивность

Проверить ширины:

- [ ] 1440px.
- [ ] 1280px.
- [ ] 1024px.
- [ ] 768px.
- [ ] 430px.
- [ ] 390px.
- [ ] 360px.

Mobile:

- [ ] Header не ломается.
- [ ] Меню открывается.
- [ ] H1 не вылезает.
- [ ] Кнопки удобно нажимать.
- [ ] Фильтры не создают горизонтальный скролл.
- [ ] Карточки в одну колонку.
- [ ] Timeline вертикальный.
- [ ] Нет горизонтального скролла.

## Accessibility

- [ ] Кнопки имеют понятный текст.
- [ ] Ссылки имеют href.
- [ ] Mobile menu доступно с клавиатуры.
- [ ] Достаточный контраст текста.
- [ ] Не использовать div вместо button для интерактива.
- [ ] У декоративных элементов aria-hidden="true".
- [ ] Радиогруппы конфигуратора доступны.
- [ ] Empty state понятен screen reader.

## Performance

- [ ] Нет тяжёлых изображений без необходимости.
- [ ] Изображения оптимизированы.
- [ ] Нет огромных библиотек ради простых эффектов.
- [ ] Анимации используют transform / opacity.
- [ ] Нет layout shift.
- [ ] Нет бесконечных дорогих JS-анимаций.
- [ ] Каталог не рендерит лишние тяжёлые элементы без необходимости.

## SEO / Metadata

- [ ] Title отражает игровые ПК в Великом Новгороде.
- [ ] Description короткий и коммерческий.
- [ ] Есть Open Graph image.
- [ ] Основной H1 один.
- [ ] Секции имеют логичные H2.
- [ ] Нет длинных SEO-простыней.

## Финальная Проверка

- [ ] Сайт сразу объясняет, что продаётся.
- [ ] Пользователь понимает, как выбрать ПК.
- [ ] Пользователь понимает, как заказать ПК.
- [ ] Пользователь видит реальные характеристики.
- [ ] Пользователь видит условия покупки.
- [ ] Пользователь может быстро написать.
- [ ] Есть ощущение premium tech.
- [ ] Сайт готов для локальной продажи игровых ПК.
