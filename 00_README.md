# TOGOSHOL — комплект MD-документов для разработки

Проект: премиальный локальный сайт для продажи игровых ПК в Великом Новгороде.

Цель: сделать frontend-часть сайта, который показывает реальные сборки, помогает подобрать ПК и быстро переводит пользователя в диалог.

## Что внутри

```text
00_README.md
01_PROJECT_BRIEF.md
02_DESIGN_SYSTEM.md
03_PAGE_STRUCTURE.md
04_COMPONENTS.md
05_ANIMATIONS.md
06_CONTENT.md
07_CODEX_TASKS.md
08_QA_CHECKLIST.md
```

## Основная идея

TOGOSHOL — локальный премиальный бренд игровых ПК в Великом Новгороде.

Сайт должен выглядеть не как обычный компьютерный магазин, а как чистая high-tech витрина в стилистике 2026 года:

- тёмный premium tech;
- cyan / blue / violet glow;
- крупная типографика;
- минимальное количество текста;
- каталог реальных сборок без перегруза;
- 3–6 рекомендуемых сборок на первом экране;
- конфигуратор под бюджет, игры и монитор;
- индивидуальный подбор;
- условия покупки и гарантия;
- понятная заявка;
- рабочие анимации;
- адаптив под desktop / tablet / mobile.

## Рекомендуемый стек

Если проект уже существует — использовать текущий стек.

Если проект пустой:

- React;
- TypeScript;
- Vite;
- CSS Modules или обычный CSS;
- Framer Motion опционально;
- без backend;
- без реальной корзины;
- без оплаты;
- без личного кабинета.

## Главный принцип

Не перегружать, но и не делать пустой декоративный лендинг.

Сайт должен продавать ощущение надёжной локальной сборки игровых ПК, показывать реальные варианты и объяснять, как заказать компьютер.
