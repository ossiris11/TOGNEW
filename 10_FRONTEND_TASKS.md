# TOGOSHOL — список задач по фронтенду

## P0 — сделать в первую очередь

- Проверить и заменить все реальные ссылки контактов: VK, Telegram, Instagram, Avito.
- Убрать публичные placeholder-отзывы. На сайте должны быть только реальные отзывы или честное скрытие блока.
- Проверить адаптив header после добавления соц-иконок: 1440, 1280, 1024, 768, 390, 360.
- Проверить каталог на mobile: фильтры, карточки, CTA, отсутствие горизонтального скролла.
- Проверить модалку “Собрать из комплектующих”: фокус, закрытие, mobile layout, длинные названия комплектующих.

## P1 — коммерческий polish

- Добавить loading skeletons для товаров, отзывов и комплектующих при обращении к API.
- Улучшить формы заявок: маска/валидация контакта, понятные success/error состояния, disabled-состояния.
- Сделать единый компонент социальных ссылок, чтобы header/footer/final CTA не расходились.
- Проверить CTA-тексты: “Консультация”, “Написать в VK”, “Оставить заявку”, “Отправить сборку”.
- Добавить настоящие фото/скрины отзывов из Avito/VK через админку.

## P2 — качество и запуск

- Прогнать Playwright/screenshot-проверку desktop/mobile.
- Проверить доступность: keyboard navigation, focus states, aria для модалок и раскрывающихся блоков.
- Оптимизировать hero image и тяжёлые изображения каталога.
- Проверить SEO/meta/OG на публичной странице.
- Удалить неиспользуемые старые компоненты и данные: `ReadyBuilds`, `BuildCard`, `PcMockup`, если они больше не нужны.
